# Equifax.bot

A credit education and dispute-prep guide with score factors, common report errors, and step-by-step correction checklists.

## Overview

Equifax.bot enables real-time credit intelligence through automated reports, instant scoring, and continuous monitoring.

## Key Features

- Automated credit reports & scoring
- Real-time credit monitoring
- Credit score simulator
- Global infrastructure overview
- Competitive comparison dashboard

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **State Management:** React Hooks

## Highlights

- Clean, modern fintech UI
- Fully responsive design
- Component-based architecture
- Scalable and production-ready layout
